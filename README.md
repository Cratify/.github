## The Future Runs Locally

Most AI tools want your data.  
**Cratify** gives you superpowers **without** sending anything anywhere.

A lightning-fast, native desktop app where you can:
- Chat with your entire filesystem
- Deploy powerful local agents
- Forge custom AI workflows
- Run the latest models at full speed

Built from scratch with **Rust + Tauri** — tiny, blazing fast, and actually beautiful.

## Features That Actually Matter

| Feature | Description | Status |
|--------|-------------|--------|
| **Native RAG** | Drag & drop folders → instantly chat with all your notes, PDFs, code, research | ✅ |
| **Agent Forge** | Build, test, and run multi-agent teams with tools (browser, terminal, code exec) | ✅ |
| **Model Studio** | One-click Ollama + llama.cpp + Candle support. Compare models side-by-side | ✅ |
| **Zero Bloat** | < 25MB install. Blazing performance. Feels like a native macOS/Windows app | ✅ |
| **Global Hotkey** | `Ctrl/Cmd + Shift + Space` — AI instantly appears | ✅ |
| **Privacy First** | Everything stays on your machine. Auditable. Open source. | ✅ |
