## The Future Runs Locally

Most AI tools want your data.  
**Cratify** gives you superpowers **without** sending anything anywhere.

A lightning-fast, native desktop app where you can:
- Chat with your entire filesystem
- Deploy powerful local agents
- Forge custom AI workflows
- Run the latest models at full speed

Built from scratch with **Rust + Tauri** — tiny, blazing fast, and actually beautiful.
